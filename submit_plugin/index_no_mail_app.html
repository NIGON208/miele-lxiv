Submit your Plugin

This page is for plugins developers!

You have a plugin and you want to share it? Or sell it? Then this page is for you. Follow the instructions bellow to find out how to get your plugin in this list.

Plugin binary Design your plugin and build it. To get you started coding, an Xcode plugin generator template is available at this URL:
https://github.com/bettar/miele-lxiv-plugin-generator
Make sure that your plugin works!

Hosting We are not providing hosting, you will have to place your plugin on one of your servers. Host it zipped.

Page You also have to host the presentation page (HTML) of your plugin that will show in this list. A template is available here: template.zip

Last step Once your plugin is built, hosted and that the presentation page is done and available on the internet, please send us an e-mail containing the following informations:

NameThe name of the plugin. Don't make it too long, it will be displayed in the top pop-up menu of this window.
VersionThe version number will be used to warn the user if he already owns this version of the plugin.
Page URLThis is the address of the presentation page (HTML) that you can use the template: template.zip
Binary URLThis is the address of the plugin binary (zipped!). This is not a required info, if you don't provide it, you will have to handle the download yourself. This possibility can be used only if you sell your plugin. Otherwise, let us handle the download & install of the plugin.

Send the e-mail to:

miele-lxiv@gmx.com

Once we receive the e-mail, your submission will be examined and go through a validation process.
