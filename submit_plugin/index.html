Submit your Plugin

This page is for plugins developers!

You have a plugin and you want to share it? Or sell it? Then this page is for you. Follow the instructions bellow to find out how to get your plugin in this list.

Plugin binary Design your plugin and build it. Make sure that it works!

Hosting We are not providing hosting, you will have to place your plugin on one of your servers. Host it zipped.

Page You also have to host the presentation page (HTML) of your plugin that will show in this list. A template is available here: template.zip

Last step Once your plugin is built, hosted and that the presentation page is done and available on the internet, please submit us the following information:

Name
Enter the name of the plugin here. Don't make it too long, it will be displayed in the top pop-up menu of this window.
Version
The version number will be used to warn the user if he already owns this version of the plugin.
Page URL
This is the address of the presentation page (HTML) that you can base it on the template: template.zip
Binary URL
This is the address of the plugin binary (zipped!). You can leave this field empty, in this case you will have to handle the download yourself. This possibility can be used only if you sell your plugin. Otherwise, let us handle the download & install of the plugin.
E-mail
Enter your e-mail address here. We will only use it if we have any questions about the plugin in our validation process.

Once you are done, submit the informations using the submit button. Your submission will be examined and go through a validation process.
